## Como executar o projeto
1→ Crie o ambiente virtual python e instale as dependencias do projeto:
```bash
conda activate ml 

python -m venv .venv
# Windows
.venv\Scripts\activate

# Linux ou MacOS
source .venv/bin/activate
pip install -r requirements.txt
```
